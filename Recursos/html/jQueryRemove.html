jQuery Remove

Eliminar elementos/contenido


  • remove() - Inserta contenido al final de los elementos seleccionados
  • empty() - Inserta contenido al principio de los elementos seleccionados





    • Ejemplo del método jQuery remove()

    Código

                   
                function agregarJQueryRemove()
                {
                  $(document).ready(function(){
                    $("button").click(function(){
                      $("#div1").remove();
                    });
                  });

    Método jQuery vacío ()


    El método jQuery empty() elimina los elementos secundarios de los elementos seleccionados.





    Ejemplo del método jQuery empty()

    Código

                
              function agregarJQueryEmpty()
              {
                $(document).ready(function(){
                  $("button").click(function(){
                    $("#div1").empty();
                  });
                });
              }

    Filtrar los Elementos a Eliminar


    El método jQuery remove() también acepta un parámetro, que le permite filtrar los elementos que se eliminarán.

    El parámetro puede ser cualquiera de las sintaxis del selector de jQuery.





    El siguiente ejemplo elimina todos los p elementos con class="test":

    Código

                
              function filtrarJQueryEmptyRemove()
              {
                $(document).ready(function(){
                  $("button").click(function(){
                    $("p").remove(".test");
                  });
                });
              }
